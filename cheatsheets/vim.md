# Hello everybody22

here is my personal vi/vim cheatsheet, have fun!

{% highlight bash %}
dd - delete lines
dNw - delete N words

x - delete single char
Nx - delete N chars

ZZ(capital) save and exit

u - undo
U (capital) - undo all changes to the current line

} - next paragraph
{ - previous paragraph

w - next word

$ - end of the current line
G - last line
{% endhighlight %}
